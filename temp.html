HTML COURSE

Lesson 1: Introduction to HTML

HTML (HyperText Markup Language) is the backbone of web pages, structuring content like text, images, and links. It uses tags to define elements, creating a tree-like structure called the DOM (Document Object Model). HTML5, the latest version, adds semantic tags and multimedia support, enabling accessible, interactive sites. Learning HTML unlocks web development’s foundation.

Code Example:

<html>
    <body>
        <p>Hello, World!</p>
    </body>
</html>

Lesson 2: Basic Structure with <html>

The <html> tag is the root of an HTML document, containing all other tags. It’s a container, typically including <head> for metadata and <body> for content. The lang attribute (e.g., lang="en") aids accessibility, telling browsers and screen readers the page’s language. It sets up every web page.

Code Example:

<html lang="en">
    <head>
        <title>My Page</title>
    </head>
    <body>
        <p>Content</p>
    </body>
</html>

Lesson 100: Real-World: Portfolio Page

A portfolio page showcases projects using <section>, <article>, and <img> for visual appeal. Semantic tags like <header> and <nav> provide structure, with links (<a>) to project details. Accessibility requires alt text and ARIA roles, like role="region", for clarity. The <footer> adds contact info, ensuring a professional touch. This example combines HTML elements for a polished, user-friendly portfolio layout.

Code Example:

<section>
    <header>
        <h1>My Portfolio</h1>
    </header>
    <article>
        <h2>Project 1</h2>
        <p>Description here.</p>
        <img src="project1.jpg" alt="Project screenshot">
    </article>
    <footer>
        <p>Contact: me@example.com</p>
    </footer>
</section>

HTML LIBRARY

<a>
Creates a hyperlink to another page or resource.
<body>
Contains the visible content of the web page.
<button>
Creates a clickable button for user interaction.
<div>
A container for grouping content or styling.
<form>
Creates a form for user input, like text or selections.
<h1>
Defines the most important heading (largest).
<head>
Contains metadata, like title and links to styles.
<html>
The root element of an HTML document.
<img>
Embeds an image in the page.
<input>
Creates an input field for user data entry.
<label>
Provides a caption for a form input.
<li>
Defines a list item in an ordered or unordered list.
<link>
Connects to external resources, like CSS files.
<meta>
Provides metadata, like character set or viewport.
<nav>
Defines a navigation section with links.
<p>
Represents a paragraph of text.
<span>
An inline container for styling small text portions.
<table>
Creates a table for displaying data in rows and columns.
<title>
Sets the page title shown in the browser tab.
<ul>
Creates an unordered (bulleted) list.
